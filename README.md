# 📌 Flask TODO API  

Простое REST API на Flask для управления задачами (TODO). Поддерживает операции CRUD (Create, Read, Update, Delete).  

## 🚀 Возможности API  
- 📌 Получение списка всех задач  
- 🔍 Получение задачи по ID  
- ✏️ Создание новой задачи  
- 🔄 Обновление задачи  
- 🗑 Удаление задачи
